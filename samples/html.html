HTML

  • Elements: these are the 'parts' of a web page e.g. in the following snippet, there are several elements such as <html> and <body> which are found in all webpages. This also has a main header using the <h1> element and a paragraph using the <p> element. 
    <html>
  <head>
    <title>My Github Pages</title>
  </head>
  <body>
    <h1>Welcome to my github pages!</h1>
    <p>A paragraph of text.</p>
  </body>
</html>
  • Attribute: these are values within specific Elements
